# Analysis of Test Coverage

## SOURCE FILES
1. MakeselfMojo.java (1157 lines)
   - Main mojo for creating makeself archives
   - Key methods:
     * execute() - Main entry point
     * execute(List<String>, boolean) - Process execution
     * extractMakeself() - Extract embedded scripts
     * setFilePermissions(File) - Set executable flag
     * setPosixFilePermissions(Path) - Set POSIX permissions
     * loadArgs() - Build command-line arguments
     * isTrue(Boolean) - Null-safe Boolean conversion

2. GitMojo.java (77 lines)
   - Downloads and installs portable Git on Windows
   - Key methods:
     * execute() - Main entry point (extends AbstractGitMojo)

3. AbstractGitMojo.java (223 lines)
   - Abstract base for Git-related operations
   - Key methods:
     * checkGitSetup() - Initializes Git setup
     * extractPortableGit() - Extracts portable Git
     * installGit(Artifact, String) - Installs Git from artifact
     * runInstaller(List<String>) - Runs installer process

4. PortableGit.java (130 lines)
   - Properties holder for portable Git configuration
   - Constructor loads from makeself.properties
   - Getters for: groupId, artifactId, version, extension, classifier, name

5. ProcessUtils.java (28 lines)
   - Empty placeholder utility class
